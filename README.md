# Driver Drowsiness Detection System

This project detects driver drowsiness using Convolutional Neural Networks (CNN) and alerts the driver when drowsiness is detected.

## Technologies Used
- Python
- Django
- OpenCV
- CNN (Convolutional Neural Networks)

## Features
- Real-time face detection
- Eye detection and monitoring
- Drowsiness alert system
- Machine learning based prediction

## Purpose
The main objective of this project is to improve road safety by detecting driver fatigue and preventing accidents.
